From fax machines to web applications

  • UX Planning

  • UI Design

GE has a commercial loan business supplying multi-million dollar loans to very well known and large companies. Historically, they manage these relationships via phone calls, emails and paper filing via fax machine. Obviously, this was a bit behind the times so they conceptualized a way to do this over a web application replacing a process over a decade old. The application itself establishes loan creation, updating and maintenance of existing agreements. From a process perspective, the application has a number of breakpoints requiring legal signoffs and points of negotiation. Once all the steps of these sign-offs are complete, loan holders can view transactions and set up requests for micro-loans. Very exciting stuff I know.
