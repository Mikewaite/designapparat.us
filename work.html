Capsule

WordPress, Web App

Capsule is the brainchild of the lovely people at Crowd Favorite. It uses WordPress to create a developers notepad. All of the entires are taggable and can be searched via names, code types and date. It's a great tool to use if you want to keep some code handy to use later. I had the pleasure to design some look and feel components of this project.

Betfair

Data Exploration

Betfair is the world's largest betting exchange. With millions of transactions everyday, they are sitting on one of the largest repositories of data ever assembled. While none of it is public, the work was a good exercise in large data sets.
